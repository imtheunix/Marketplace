Afinal, o que é BusuTBM?

BusuTBM é uma plataforma de mercado para negociação de robôs automatizados para trading, crie uma conta e explore máquinas criadas por outros usuários. Todos os códigos são auditados antes de serem catalogados. A qualidade e relevância do seu anúncio será determinada pela classificação de 1 a 5 estrelas feita pelo comprador.

Django

BusuCMR utiliza Django, um framework para Python provado ser um dos mais seguros e escaláveis, ele utiliza de módulos para gerar páginas, então tudo o que você vê neste site, está sendo renderizado no servidor, e não diretamente no navegador.

Github

Todas as modificações de código são opensouce, ou seja, todo o código base da plataforma esta disponível para todos, você pode ajudar a melhorar o código fazendo pull requests, ou apenas fazer um commit, dizendo o que você deseja implementar ou remover.
